# Sokoban Game written in C++11

## Dependencies
- [Boost](https://www.boost.org/)

## Build Tools:
- [CMake](https://cmake.org/)
- [Autotools](https://www.gnu.org/software/software.html)

## Building Instructions:

### Autotools
1. mkdir build
2. cd build
3. autoreconf -fiv ../
4. ../configure
5. make

### CMake
1. mkdir build
2. cmake ../
3. make
